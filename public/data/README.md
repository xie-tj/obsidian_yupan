# 接入真实 A 股历史日线（盲盒训练）

默认 `manifest.json` 为空，应用使用**合成行情**（已叠加隔夜跳空、连板延续、量价耦合等 A 股结构）。
想用真实历史把这里填上即可——盲盒会随机抽取其中一只，盘中隐藏名称/日期，**结算时才揭晓**。

## 步骤

1. 准备 CSV，表头固定、按日期升序、**至少 200 行**（预热 160 根后才有足够可交易区间）：

   ```csv
   date,open,high,low,close,volume
   2021-01-04,1998.00,2050.00,1995.10,2040.00,3120000
   2021-01-05,2041.00,2065.00,2010.00,2025.50,2890000
   ...
   ```

   - 价格为**前复权**价（保证涨跌停撮合逻辑成立）。
   - `volume` 单位随意（仅用于副图相对高低）。

2. 把 CSV 放进本目录（`public/data/`）。

3. 在 `manifest.json` 登记：

   ```json
   [
     { "file": "600519.csv", "name": "贵州茅台", "code": "600519" },
     { "file": "300750.csv", "name": "宁德时代", "code": "300750" }
   ]
   ```

   - `code` 前缀决定板块与涨跌停幅度：`600/000→±10%`、`300/688→±20%`、`8/43/92→±30%`。

## 数据从哪来

- 券商客户端「导出历史数据」
- 开源库：[AKShare](https://akshare.akfamily.xyz/)、[Tushare](https://tushare.pro/) —— 拉日线后导出为上面的列即可。

> 仓库默认不附带真实行情数据（版权/时效）。合成行情已足够练手；要贴近实盘，按上面填入你自己的历史段。
